◆ update_grid_counts()

void Grid::update_grid_counts ( std::vector< Person > &  pop)

Function which creates the the vector of people currently located at each point on the grid.

Note that the way I did this was to first call reset_grid to set all counts to zero. Then I simply iterate through each person in the pop vector and push_back the relevant population vector index to the data vector. i.e. pop is the vetor with each element being a Person. data[i][j] contains a vector of Persons currently at coordinate (i,j). Suppose pop[7] is at location (2,3). Then you could push_back integer "7" to data[2][3] for this example. Just be consistent with what order you store the x and y variables.

Parameters
popvector of people

◆ update_health_status()

void Grid::update_health_status ( std::vector< Person > &  pop)

Function to update the health status of the population.

This loops over each Person at the end of each turn and changes the counters and status. For example, if the person is in state "infected" and days_to_recovery goes to zero, then the person becomes "immune". If days_to_death goes to zero, they become dead and are removed from the population vector. You can user the vector erase(it) function however you will need to figure out how to do the loop. erase() returns the next iterator for the sequence. If you erase the last element, erase returns vector.end(). However, if you just have a naive for loop with ++it, when the loop comes around again you will "increment" past the end of the vector.
