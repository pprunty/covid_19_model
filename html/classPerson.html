void Person::infect ( )
private

Function to "infect" a particular individual.

This function set the persons "infected" status to true. It will also calculate the days_to_death as as exponentially distributed random variable. the parameter of the expenential distribution being determined by percentile matching so that "death_percentage" occurs by "time_to_recovery". See assignment document

◆ propose_move()

void Person::propose_move ( )

Function to propose, and then accept or reject a move.

This function will propose a move from current_location to a new location. See assignment document for rules. The Person can remain in it's current location or else more to one of the 8 adjacent points.

Use Mettropolis algorithm to decide whether or not to accept a move.
